# 🩺 INSULIX

Sistema inteligente para el monitoreo y control de la diabetes mediante Machine Learning, integración con sensores CGM y generación de reportes médicos automatizados.

## 🚀 Tecnologías

- Ionic
- Angular
- Node.js
- MongoDB
- Firebase
- Machine Learning

## 📋 Características

- Análisis predictivo
- Alertas y recordatorios
- Integración CGM
- Reportes médicos
- Sincronización en la nube
